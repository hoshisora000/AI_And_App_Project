## 功能簡介
透過連線自訂API來實作發票APP的各項功能

1.會員登入

2.取得資料庫內發票內容

3.手動輸入發票

4.掃描電子發票QR code 

5.AI辨傳統發票

6.消費分析

7.API開發與應用
