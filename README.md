## 功能簡介

1.會員登入(基於firebase進行登入，並於資料庫儲存會員資料)

2.雲端資料庫存儲發票

3.手動輸入發票

4.電子發票QR code快速輸入 

5.傳統發票AI辨識輸入

6.發票兌獎功能

7.消費分析

8.API開發與應用

9.網站開發與管理員後台介面
